# Inala

> Inala responsive website

## Technologies

- [**Gulp**](http://gulpjs.com)
- [**Html**](https://developer.mozilla.org/es/docs/HTML/HTML5) 
- [**Sass**](http://sass-lang.com)  
- [**Babel**](https://babeljs.io)

## Install and Use

### Install

```bash
npm install -g Inala
```

### Use 

```bash
yo Inala
```
